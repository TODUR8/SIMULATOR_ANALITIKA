Описание проекта.

Новый стартап который активно развивает приложение на мобильных телефонах.

Шаг 1.
Построим базовый дашборд которые покрывает основные аналитические запросы компании.  И отвечают на первый базовый
вопрос сколько? Основной дашборд это такая штука которая нам позволяет нам сразу же посмотреть какието то супер
базовые вещи которые должны быть всегда на глазах. 

Рассчитаем DAU, WAU и MAU. Вычислим средние значения этих метрик за весь период. Отобразим изменения метрик 
во времени на графиках.


Шаг 2.
Построим график Retention.
Построим график новых, старых и ушедших пользователей по неделям.

Шаг 3.
С помощью А/B-теста оценим новый алгоритм рекомендации постов.

Шаг 4.
Сформулируем выводы.


Описание данных:

Структура df_action
* user_id - id пользователя
* post_id - id поста
* action - действие пользователя
* time - время действия

Структура df_log
* user_id - id пользователя
* gender - пол
* age - возраст
* country - страна
* city - город
* os - операционная система
* source - источник пользователя
* exp_group - принадлежность к тестовой/эксперементальной группе
